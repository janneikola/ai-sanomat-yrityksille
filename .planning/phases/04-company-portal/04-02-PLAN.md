Portal API routes (registered at /api/portal/*):
```
POST /api/portal/login    body: { email: string }           -> { message: string }
POST /api/portal/verify   body: { token: string }           -> { success: true, redirectUrl: '/tiimi' } | 401
GET  /api/portal/members                                    -> Member[]
POST /api/portal/members  body: { email: string, name?: string } -> Member
POST /api/portal/members/bulk body: { emails: string }      -> { added: number, reactivated: number, skipped: number, invalid: string[] }
PATCH /api/portal/members/:id                               -> { success: true }
GET  /api/portal/archive                                    -> Issue[] (sent only)
GET  /api/portal/me                                         -> { companyName: string, plan: string, contactEmail: string }
POST /api/portal/logout                                     -> { success: true }
```

Member shape:
```typescript
{ id: number; email: string; name: string | null; isActive: boolean; isBounced: boolean; clientId: number; createdAt: string; updatedAt: string }
```

Issue (archive) shape:
```typescript
{ id: number; weekNumber: number; year: number; status: 'sent'; createdAt: string; updatedAt: string }
```

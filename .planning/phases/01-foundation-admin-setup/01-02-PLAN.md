API Endpoints (from Plan 01):
- POST /api/auth/login — { email, password } → sets httpOnly cookie, returns { success: true }
- POST /api/auth/logout — clears cookie
- GET /api/auth/me — returns { email, role }
- GET /api/admin/clients — list all clients
- POST /api/admin/clients — create client
- GET /api/admin/clients/:id — get single client
- PUT /api/admin/clients/:id — update client
- GET /api/admin/sources — list all sources
- POST /api/admin/sources — create source
- PUT /api/admin/sources/:id — update source
- PATCH /api/admin/sources/:id/toggle — toggle active
- GET /api/admin/templates — list all templates
- GET /api/admin/templates/:id — get single template
- PUT /api/admin/templates/:id — update template
